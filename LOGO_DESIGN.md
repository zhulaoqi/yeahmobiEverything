## 🎨 视觉设计

### 配色方案

基于 UI/UX 最佳实践（Soft UI Evolution）：

```
主色：#3B82F6 (Trust Blue)     - 信任、专业、科技
副色：#60A5FA (Light Blue)     - 亲和、清晰、现代
强调色：#2563EB (Deep Blue)    - 眼睛和鼻子
背景：渐变蓝色                  - 冰原意象
```

### 形状设计

**简洁轮廓风格：**
- ✅ 使用基本几何形状（椭圆、圆形）
- ✅ 白色填充（#FFFFFF），不透明度 235-245
- ✅ 圆润边缘，无尖角
- ✅ 比例协调，易于识别

**元素组成：**
```
头部（圆形）    ← 大圆形
  ├─ 耳朵（2个）  ← 小圆形
  ├─ 眼睛（2个）  ← 蓝色圆点
  └─ 鼻子（1个）  ← 蓝色椭圆

身体（椭圆）    ← 大椭圆
  ├─ 手臂（2个）  ← 长椭圆
  └─ 脚（2个）    ← 宽椭圆
```

### 尺寸规格

## 📐 技术规格

### 文件结构

```
everything-client/src/main/resources/images/
├── app-icon.png          ← 主图标（1024x1024）
├── app-icon.icns         ← macOS 图标集
├── app-icon-preview.png  ← 预览图（256x256）
└── app-icon-e-version.png ← 旧版 E 字母（备份）
```

### .icns 图标集包含

```
icon_16x16.png        icon_16x16@2x.png
icon_32x32.png        icon_32x32@2x.png
icon_128x128.png      icon_128x128@2x.png
icon_256x256.png      icon_256x256@2x.png
icon_512x512.png      icon_512x512@2x.png
```

### 生成方法

```bash
# 1. 使用 Python PIL 生成 PNG
python3 create_logo.py

# 2. 使用 macOS sips 和 iconutil 生成 .icns
sips -z [size] [size] app-icon.png --out icon_[size].png
iconutil -c icns app-icon.iconset
```

## 🛠️ 技术实现

### 生成工具
- **Python PIL (Pillow)**：绘制矢量风格图形
- **macOS sips**：批量缩放
- **macOS iconutil**：生成 .icns

### 核心代码
```python
# 头部（圆形）
draw.ellipse(
    [(center - head_radius, head_y - head_radius),
     (center + head_radius, head_y + head_radius)],
    fill=(255, 255, 255, 245)
)

# 眼睛（蓝色圆点）
draw.ellipse(
    [(center - eye_offset_x - eye_radius, eye_y - eye_radius),
     (center - eye_offset_x + eye_radius, eye_y + eye_radius)],
    fill=(59, 130, 246, 255)  # Trust Blue
)
```
